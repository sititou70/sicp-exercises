## 依存

- build-essential (12.9ubuntu3)
- libcunit1-dev (2.1-3-dfsg-2.4)
- make

## 実行方法

### repl

```sh
make
./main
```

### ファイルを実行

```sh
make
./main sample.rkt
```

## テスト

```sh
make test
```

## 生成物を削除する

```sh
make cleam
```
