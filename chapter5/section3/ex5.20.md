## `(define x (cons 1 2))`を実行したとき

箱とポインタ表現：

```text
x -> [.|.]
      | |
      v v
      1 2
```

メモリベクタ表現：

| Index | the-cars | the-cdrs | 備考    |
| :---: | :------: | :------: | :------ |
|   0   |          |          |         |
|   1   |    n1    |    n2    | <- x    |
|   2   |          |          | <- free |

## 続けて`(define y (list x x))`を実行したとき

箱とポインタ表現：

```text
x -> [.|.]
      | |
      v v
      1 2

y -> [.|.]-> [.|/]
      |       |
      v       v
      x       x
```

メモリベクタ表現：

| Index | the-cars | the-cdrs | 備考    |
| :---: | :------: | :------: | :------ |
|   0   |          |          |         |
|   1   |    n1    |    n2    | <- x    |
|   2   |    p1    |    p3    | <- y    |
|   3   |    p1    |    e0    |         |
|   4   |          |          | <- free |
