z1 と z2 の構造を以下に示す。

```text
z1 -> [.|.]
       | |
       v v
x --> [.|.]-> [.|/]
       |       |
       v       v
       a       b
```

```text
z2 -> [.|.]-> [.|.]-> [.|/]
       |       |       |
       |       v       v
       |       a       b
       |       ^       ^
       |       |       |
       +----> [.|.]-> [.|/]
```

`set-to-wow!`手続きは、引数の caar を`'wow`へのポインタに置き換える。

z1 に適用すると、

```text
z1 -> [.|.]
       | |
       v v
x --> [.|.]-> [.|/]
       |       |
wow <--+       |
               v
       a       b
```

となり、z1 は`((wow b) wow b)`となる。

一方、z2 に適用すると、

```text
z2 -> [.|.]-> [.|.]-> [.|/]
       |       |       |
       |       v       v
       |       a       b
       |               ^
       |               |
       +----> [.|.]-> [.|/]
               |
               v
              wow
```

となる。したがって、z2 は`((wow b) a b)`となる。
